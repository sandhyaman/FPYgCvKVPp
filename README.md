# 学生信息管理系统 python920

## 项目概述

学生信息管理系统是基于 Python 开发的一个数据库驱动的应用，旨在帮助学校或教育机构高效管理学生信息。系统具备完善的功能模块，支持数据的增删改查等基本操作，并包含登录注册机制以及错误处理机制。

## 技术栈

- **后端**: Python
- **数据库**: MySQL

## 功能模块

### 基础功能

- 学生信息的增加、删除、修改和查询
- 成绩的录入与修改，且成绩范围限制在 0-100 分之间

### 数据分析

- 成绩统计功能，以柱状图形式展示
- 筛选不同成绩区间内的学生信息

### 排序与筛选

- 多条件排序功能

### 用户认证

- 登录与注册功能

### 错误处理

- 对各种操作中的异常进行捕获与处理

## 系统角色

- 系统管理员
- 教师用户
- 学生用户

## 运行环境

- Python 环境配置
- MySQL 数据库服务器

## 部署步骤

1. 安装 Python 环境
2. 安装 MySQL 数据库
3. 导入 SQL 文件以初始化数据库
4. 配置系统参数
5. 启动应用程序
